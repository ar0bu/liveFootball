# liveFootball

### Installation

```sh
$ npm install
$ bower install
$ sudo npm install -g cordova
$ sudo npm install -g ionic
$ cordova platform add android
```

### Run on browser
```sh
$ ionic serve --lab
```

### Build android application
```sh
$ ionic build android
```

### Run use Android emulator
```sh
$ ionic emulate android
```
